# Conch Shell

A simple shell for <a href="https://www.holbertonschool.com/">Holberton School</a>.

## Overview

Conch is a simple implementation of a shell in C.

##Examples

Run a built-in:

`/bin/ls`

Or:

`ls`

To print enviroment:

`env`

To Exit:

`exit`
